Дан двумерный массив (матрица), состоящий из N строк и N элементов, где N вводится с клавиатуры при помощи prompt.
Заполните каждую строку случайными числами в диапазоне от -10 до 30. Определите:
- количество строк, не содержащих ни одного нулевого элемента;
- максимальное из чисел в заданном массиве.
